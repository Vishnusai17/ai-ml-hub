## 📄 Research Papers

> See [`research-papers/`](./research-papers/) for full summaries and detailed paper reviews.

### 🏛️ Landmark Papers (Must-Read)
| Year | Paper | Topic |
|---|---|---|
| 2017 | [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | Transformers |
| 2018 | [BERT](https://arxiv.org/abs/1810.04805) | NLP Pre-training |
| 2020 | [GPT-3](https://arxiv.org/abs/2005.14165) | Large Language Models |
| 2021 | [ViT](https://arxiv.org/abs/2010.11929) | Vision Transformers |
| 2022 | [DALL-E 2](https://arxiv.org/abs/2204.06125) | Text-to-Image |
| 2023 | [LLaMA](https://arxiv.org/abs/2302.13971) | Open LLMs |

### 🚀 Emerging Trends (2024–2025)
| Paper | Topic | Summary |
|---|---|---|
| [Mamba](https://arxiv.org/abs/2312.00752) | State Space Models | Efficient alternative to Transformers |
| [Mixtral](https://arxiv.org/abs/2401.04088) | Mixture of Experts | Sparse MoE for efficient LLMs |
| [Sora](https://openai.com/sora) | Video Generation | World-model video synthesis |
| [DeepSeek-R1](https://arxiv.org/abs/2501.12948) | Reasoning LLMs | Chain-of-thought reasoning at scale |
